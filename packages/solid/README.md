# @fluix-ui/solid

Solid adapter for Fluix UI components.

## Install

```bash
npm install @fluix-ui/solid @fluix-ui/css
```

## Usage

```ts
import { fluix } from "@fluix-ui/solid";
import "@fluix-ui/css";
```
