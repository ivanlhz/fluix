# @fluix-ui/css

Shared CSS for Fluix UI components.

Uses the Fluix data-attribute contract (`data-fluix-*`) and provides animations, variables, and toast styles.

## Install

```bash
npm install @fluix-ui/css
```

## Usage

```ts
import "@fluix-ui/css";
```
