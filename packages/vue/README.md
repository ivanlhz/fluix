# @fluix-ui/vue

Vue 3 adapter for Fluix UI components.

## Install

```bash
npm install @fluix-ui/vue @fluix-ui/css
```

## Usage

```ts
import { Toaster, fluix } from "@fluix-ui/vue";
import "@fluix-ui/css";
```
