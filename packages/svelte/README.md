# @fluix-ui/svelte

Svelte 5 adapter for Fluix UI components.

## Install

```bash
npm install @fluix-ui/svelte @fluix-ui/css
```

## Usage

```ts
import { Toaster, fluix } from "@fluix-ui/svelte";
import "@fluix-ui/css";
```
