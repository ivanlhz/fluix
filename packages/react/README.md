# @fluix-ui/react

React adapter for Fluix UI components.

## Install

```bash
npm install @fluix-ui/react @fluix-ui/css
```

## Usage

```tsx
import { Toaster, fluix } from "@fluix-ui/react";
import "@fluix-ui/css";
```
