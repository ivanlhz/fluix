# @fluix-ui/vanilla

Vanilla JS adapter for Fluix UI components.

## Install

```bash
npm install @fluix-ui/vanilla @fluix-ui/css
```

## Usage

```ts
import { createToaster, fluix } from "@fluix-ui/vanilla";
import "@fluix-ui/css";
```
