# Security Policy

## Supported Versions

Fluix is currently in early development (`0.x`).
Security fixes are applied on the latest code in `main`.

## Reporting a Vulnerability

Please do not open public issues for security vulnerabilities.

If you do not yet have a private security contact configured, open an issue
titled `SECURITY: private report request` without technical exploit details.
Maintainers will move to a private channel for full disclosure.

Include:
- affected package and version
- reproduction steps
- impact assessment
- any suggested mitigation

We will acknowledge receipt as soon as possible and coordinate a fix/release.
